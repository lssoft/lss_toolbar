2 Faces + Path Tool

Generate gradual morphing from one face to another and distribute results along a given path

Tool Operation

  1. open tool dialog
  2. pick first face
  3. pick second face
  4. pick path curve
  5. adjust settings
  6. apply settings to generate '2 Faces + Path' object

Advanced Operations

  • use speed buttons inside tool dialog to flip/rotate first or second face (it helps to achieve more smooth morphing result)
  • use speed button to reverse path curve if needed

Post Processing

Modify any of two initial faces and/or path curve to rebuild result object.

Note

All objects, generated using 'LSS Toolbar' has common post processing capabilities:
  • 'Refresh' command rebuilds generated results manually
  • 'Observe Changes' command enables instant refreshing after move/scale events
  • 'Properties' command provides re-adjustment of initially entered settings any time later after object creation
View 'Commands' section of LSS Toolbar 'Help Index' for more information about post processing capabilities.

Tools
 2 Faces + Path Generate gradual morphing from one face to another and distribute results along a given path
Tool processing:
  • open tool dialog
  • pick first face
  • pick second face
  • pick path curve
  • adjust settings
  • apply settings to generate '2 Faces + Path' object

Post processing:
modify any of two initial faces and/or path curve to rebuild result object.

 Blend Generate gradual morphing from one entity to another
Tool processing:
  • open tool dialog
  • pick first entity
  • pick second entity
  • adjust settings (first of all desired 'morphing' steps count)
  • apply settings to generate 'Blend' object

Post processing:
modify any of two initial entities to rebuild result object.
Learn more...
 Recursively Smoothed Curve Smooth a given curve or draw a new one
Tool processing:
  • open tool dialog
  • pick curve to be smoothed
  • adjust settings
  • apply settings to generate recursively smoothed curve
It is also possible to draw new curve interactively instead of picking an existing non-smoothed curve.

Post processing:
move curve nodal points to redraw curve.
Learn more...
 Make 3D Mesh Draw curved 3D mesh interactively
Tool processing:
  • open tool dialog
  • adjust settings
  • draw nodal points of the surface interactively
  • apply or double-click the last nodal point to generate smoothed 3D surface

Post processing:
move surface nodal points to rebuild generated surface.
Learn more...
 Control Points Deform geometry group by moving attached control points
Tool processing:
  • open tool dialog
  • pick geometry group to deform
  • add some control points interactively
  • drag control points to view deformations (optional step, since it is possible to interact with control points any time later)
  • adjust settings, apply settings to generate deformed group

Post processing:
interact with generated control points for further deformation of generated group.
Learn more...
 Stick Group Stick geometry group to any surface in an active model
Tool processing:
  • open tool dialog
  • pick geometry group to stick
  • adjust settings
  • apply settings to generate sticked group based on picked one

Post processing:
interact with initially picked group to rebuild sticked group.
Learn more...
 Follow Edges Pick face to follow edges of selected geometry group
Tool processing:
  • open tool dialog
  • pick geometry group to follow its edges
  • pick face to follow group's edges
  • pick joint component (optional step)
  • adjust settings (such as joint type, joint size, offset from surface and alignment options)
  • apply settings to generate 'Follow Edges' object

Post processing:
modify picked group, picked face or joint component (if any) to rebuild results.
Learn more...
 Voxelate Voxelate selected geometry group
Tool processing:
  • open tool dialog
  • pick geometry group to voxelate
  • pick voxel component (optional step)
  • adjust settings (first of all voxel size and voxel type)
  • apply settings to generate voxelated object based on picked group

Post processing:
modify picked group to rebuild generated group of voxels.
Learn more...
 Make Recursive Make selected group recursively defined
Use this tool to generate simple fractals.
Tool processing:
  • open tool dialog
  • pick geometry group to make it recursively defined
  • set recursion depth
  • apply settings to generate recursively defined group from picked one

Post processing:
interact with generated recursively defined group to achieve desired results (open generated group, find and select nested group in it, then move/rotate/scale/copy nested group and take a look at results).
Learn more...

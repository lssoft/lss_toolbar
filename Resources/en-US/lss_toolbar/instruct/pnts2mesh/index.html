Make 3D Mesh Tool

Draw curved 3D mesh interactively

Tool Operation

  1. open tool dialog
  2. adjust settings
  3. draw nodal points of the surface interactively
  4. apply settings or double-click the last nodal point to generate smoothed 3D surface

Advanced Operations

  • Right-click = pop-up menu with surface drawing additional commands
  • Double-click = enter last nodal point and generate the result surface

It is also possible to move nodal points to another positions while 'Make 3D Mesh' tool is active:
  1. place mouse pointer over a nodal point
  2. click and drag to move it
  3. release mouse button to leave nodal point at a new desired position
It is an optional usage scenario, since it is possible to interact with surface nodal points any time later.

Modifier Keys

  • Esc = cancel surface drawing
  • Del = delete last entered nodal point or delete particular nodal point, when mouse pointer is over it

Post Processing

Move surface nodal points to rebuild generated surface.

Note

All objects, generated using 'LSS Toolbar' has common post processing capabilities:
  • 'Refresh' command rebuilds generated results manually
  • 'Observe Changes' command enables instant refreshing after move/scale events
  • 'Properties' command provides re-adjustment of initially entered settings any time later after object creation
View 'Commands' section of LSS Toolbar 'Help Index' for more information about post processing capabilities.

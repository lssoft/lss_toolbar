Stick Group Tool

Stick geometry group to any surface in an active model

Tool Operation

  1. open tool dialog
  2. pick geometry group to stick
  3. adjust settings
  4. apply settings to generate sticked group based on picked one

Advanced Operations

  • Draw custom sticking/bouncing direction by clicking an appropriate button within tool dialog
  • Use 'Shred before sticking' option to perform more accurate sticking

Modifier Keys

  • Esc = cancel group picking

Post Processing

Interact with picked group to rebuild sticked group.

Note

All objects, generated using 'LSS Toolbar' has common post processing capabilities:
  • 'Refresh' command rebuilds generated results manually
  • 'Observe Changes' command enables instant refreshing after move/scale events
  • 'Properties' command provides re-adjustment of initially entered settings any time later after object creation
View 'Commands' section of LSS Toolbar 'Help Index' for more information about post processing capabilities.

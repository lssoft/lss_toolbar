Refresh Command

Brief

Refreshes created 'LSS Object(s)'

Usage

  1. select any entity, which is a part of an object created using any tool of 'LSS Toolbar'
  2. click command button of 'LSS Toolbar' or choose 'Refresh' command from 'Sketchup Main Menu>>Plugins>>LSS Tools'

It is possible to select multiple entities at once, then launch 'Refresh' command. Command iterates through a set of selected entities first and performs refreshing only if some of entities related to 'LSS Toolbar' were detected in selection.

Note

The result of any 'LSS Toolbar' tool processing is not just a set of new geometric entities but an object, which allows interaction after tool processing completion.
Each 'LSS Object' has two basic parts:
  • 'passive' part — new geometric entities (a direct result of tool processing)
  • 'active' part — existing (initial) and/or newly generated (utility) entities
So it is possible to continue interaction with an object by modifying its 'active' part.
There are two ways to refresh object's 'passive' part after 'active' part modification:
  • manual — 'Refresh' command
  • automatic — 'Observe Changes' command

Follow Edges Tool

Pick face to follow edges of selected geometry group

Tool Operation

  1. open tool dialog
  2. pick geometry group to follow its edges
  3. pick face to follow group's edges
  4. pick joint component (optional step)
  5. adjust settings (such as joint type, joint size, offset from surface and alignment options)
  6. apply settings to generate 'Follow Edges' object

Modifier Keys

  • Esc = cancel previously launched command (group/face/joint component picking)

Post Processing

Modify picked group, picked face or joint component (if any) to rebuild results.

Note

All objects, generated using 'LSS Toolbar' has common post processing capabilities:
  • 'Refresh' command rebuilds generated results manually
  • 'Observe Changes' command enables instant refreshing after move/scale events
  • 'Properties' command provides re-adjustment of initially entered settings any time later after object creation
View 'Commands' section of LSS Toolbar 'Help Index' for more information about post processing capabilities.

Control Points Tool

Deform geometry group by moving attached control points

Tool Operation

  1. open tool dialog
  2. pick geometry group to deform
  3. add some control points interactively
  4. drag control points to view deformations (optional step, since it is possible to interact with control points any time later)
  5. adjust settings
  6. apply settings to generate deformed group

Advanced Operations

  • Pick and drag control point = deformation
  • Double-click = enter last control point and generate deformed group
  • Right-click = pop-up menu with control points additional commands
The above operations available while adding control points to picked group goes on.

Modifier Keys

  • Esc (while adding control points to picked group) = cancel adding control points
  • Del = delete last entered control point or delete particular control point, when mouse pointer is over it
  • Ctrl = display deformation instantly while dragging entered control points or adding new one
The above keys available while adding control points to picked group goes on.

Post Processing

Interact with generated control points for further deformation of generated group.

Note

All objects, generated using 'LSS Toolbar' has common post processing capabilities:
  • 'Refresh' command rebuilds generated results manually
  • 'Observe Changes' command enables instant refreshing after move/scale events
  • 'Properties' command provides re-adjustment of initially entered settings any time later after object creation
View 'Commands' section of LSS Toolbar 'Help Index' for more information about post processing capabilities.

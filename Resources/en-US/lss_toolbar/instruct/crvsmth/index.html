Recursively Smoothed Curve Tool

Smooth a given curve or draw a new one

Tool Operation

  1. open tool dialog
  2. pick curve to be smoothed
  3. adjust settings
  4. apply settings to generate recursively smoothed curve
It is also possible to draw new curve interactively instead of picking an existing non-smoothed curve.

Advanced Operations

  • Right-click (while drawing new curve) = pop-up menu with curve drawing additional commands

Modifier Keys

  • Esc (while drawing new curve) = cancel drawing new curve

Post Processing

Move curve nodal points to redraw curve.

Note

All objects, generated using 'LSS Toolbar' has common post processing capabilities:
  • 'Refresh' command rebuilds generated results manually
  • 'Observe Changes' command enables instant refreshing after move/scale events
  • 'Properties' command provides re-adjustment of initially entered settings any time later after object creation
View 'Commands' section of LSS Toolbar 'Help Index' for more information about post processing capabilities.

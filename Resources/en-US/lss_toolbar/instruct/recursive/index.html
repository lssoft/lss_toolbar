Make Recursive Tool

Make selected group recursively defined. Use this tool to generate simple fractals.

Tool Operation

  1. open tool dialog
  2. pick geometry group to make it recursively defined
  3. set recursion depth
  4. apply settings to generate recursively defined group from picked one

Post Processing

Interact with generated recursively defined group to achieve desired results (open generated group, find and select nested group in it, then move/rotate/scale/copy nested group and take a look at results).

Note

All objects, generated using 'LSS Toolbar' has common post processing capabilities:
  • 'Refresh' command rebuilds generated results manually
  • 'Observe Changes' command enables instant refreshing after move/scale events
  • 'Properties' command provides re-adjustment of initially entered settings any time later after object creation
View 'Commands' section of LSS Toolbar 'Help Index' for more information about post processing capabilities.
